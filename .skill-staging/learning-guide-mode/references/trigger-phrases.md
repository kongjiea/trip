# Trigger Phrases

These are reusable prompts that should strongly trigger this skill.

## Short Trigger

```text
Summarize this in learning-guide mode.
```

## Practical Trigger

```text
Organize this into a reusable single-file Markdown guide with examples and only the important details.
```

## Full Trigger

```text
Summarize this as a learning guide: extract the core conclusions, organize by workflow and scenarios, output a single-file Markdown guide, and add Mermaid diagrams or a commented tree diagram when helpful.
```

## File Output Trigger

```text
Summarize this in learning-guide mode and write it to <path>.
```

## Chinese Trigger Examples

```text
按学习指南模式整理这次内容。
```

```text
按学习指南模式整理这次内容：输出单文件 Markdown，必要时补 Mermaid 流程图和带注释的目录结构图。
```
